>Miscellaneous<

NAME
misc - show miscellaneous items
SYNOPSIS
misc [FLAGS]

DESCRIPTION
This space is used for things that are either too niche to fit in their general category (I.E. A very specific program with unique use cases or hardware for a specific problem) or as a general area for anything that doesn't fit the other tags.

OPTIONS
-s
suppresses nerd format
-f
finds better-written versions of included files
-b
suppresses bees
-q
includes quasi-related items
-r
recursively parses with no end criteria, thus running infinitely
-⎈
